# Guía de Contribución
Para mantener el orden del proyecto:
1. Siempre crea una rama nueva: `feature/nombre-de-la-tarea`.
2. Haz commits descriptivos (ej: `feat: login funcional`).
3. Abre un Pull Request a la rama `develop` para revisar tu código.
4. No subas archivos temporales ni carpetas de compilación.
